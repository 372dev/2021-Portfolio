프로젝트 기간 기획:2020년 12월 8일~2021년 1월 10일
구현:2021년 1월 4일~2021년 1월 25일
프로젝트 팀 원더 디벨로퍼즈(4인)
활용한 기술 Java, Oracle DB, JDBC,
Bootstrap, jQuery, git, GitHub,
HTML, CSS, JavaScript

  1. 프로젝트 개요

  2. KH 정보교육원 응용 SW 개발자 과정중 진행한 세미 프로젝트입니다.
  3. 채식주의 레시피를 공유하는 서비스를 목표로 구현하였습니다.
  4. 레시피 랭킹 조회, 댓글 작성과 별점 등록, 레시피 검색 및 필터링, 멤버 기능과 고객센터 등 다양한 기능을 구현하였습니다.
  5. 저는 팀원의 직무를 맡았으며 책임 핵심 기능은 멤버 관련 기능 입니다. 담당 업무 상세는 아래와 같습니다.
  6. 기획단계 : 주제선정, UI설계, DB설계, 클래스 다이어그램
  7. 구현단계 : 헤더/푸터, 네이게이션 바(Bootstrap), 회원가입/로그인(해쉬/솔트/스트레칭 활용), 비밀번호 변경/회원정보 수정/회원 탈퇴(CRUD), 기타 정적 페이지 구현(채식주의 소개)
  8. 교육과정에서 배운 자바 MVC2 패턴과 JDBC를 활용하여 프로젝트를 성공적으로 마쳤습니다.

Page 1

  1. 헤더/푸터, 네비게이션 바

  2. 페이지 상단, 하단에 include 되는 헤더/푸터입니다. 헤더에는 네비게이션 바가 포함됩니다. 브랜드 로고 혹은 브랜드 이름을 클릭할 경우 메인페이지로 이동합니다.
  3. 헤더에 포함된 네비게이션 바는 부트스트랩을 활용하여 구연했습니다. 메뉴의 내용은 로그인 상태에 따라 변화하여 로그인 되어있는 경우 마이페이지가 표시되고, 로그인 되어있지 않은 경우 회원아이디, 로그아웃 버튼은 로그인, 회원가입 버튼으로 대체됩니다.
  4. 마이페이지 메뉴는 마우스를 올리면 드랍다운 메뉴가 펼쳐져 세부 기능에 간편한 접근이 가능하도록 하였습니다.

  5. 활용한 기술

  6. Java, jQuery, Bootstrap

Page 2

  1. 로그인 모달창

  2. 헤더에 포함된 로그인 버튼을 클릭 시 모달창이 열리고 아이디와 비밀번호를 입력하여 곧바로 로그인이 가능합니다. 모달창은 “아직 회원이 아닌가요?” 버튼도 포함하고 있어서 회원가입 페이지로 화면이동 또한 가능합니다.
  3. 부트스트랩 라이브러리를 이용하여 모달창을 구현하였습니다. 로그인시 별도의 페이지로 이동하는 대신 어느 페이지에서든 로그인과 로그아웃이 가능하도록 했습니다.
  4. 입력받은 아이디와 비밀번호는 서블릿을 통해 DB에서 조회되며 아이디와 비밀번호가 일치하고 회원자격 상태가 유효하다면 회원정보를 담은 객체가 session에 저장되어 로그인 상태가 유지됩니다.
  5. 회원이 아닌 경우 “아직 회원이 아닌가요?” 버튼 혹은 메뉴의 회원가입 버튼을 통해서 회원가입 페이지로 이동하여 가입이 가능하도록 하였습니다.
  6. 회원가입 시 선호하는 채식 종류의 선택이 가능하도록 하였으며, 이 정보를 활용하여 다른 페이지에 구현된 레시피 추천이 이루어 집니다.

  7. 활용한 기술

  8. Java, jQuery, Bootstrap, JDBC, Oracle DB

Page 3

  1. 회원정보 수정/비밀번호 변경/회원 탈퇴 등

  2. 마이페이지에는 마이페이지홈(회원정보 표시), 회원정보 수정, 비밀번호 수정, 회원탈퇴 등의 기능이 제공되며 이들중 민감한 정보를 다루는 기능은 로그인이 되어있더라도 현재 비밀번호를 입력해야 수정/변경이 가능하도록 했습니다.
  3. 비밀번호의 경우 해시, 솔트, 스트레칭을 이용하여 보안성을 향상했습니다. 해시 함수는 SHA256을 이용하였으며, 솔트값은 16비트의 해시 값을 회원가입 시 랜덤으로 생성하여 회원 테이블에 저장되도록 하였습니다.

  4. 활용한 기술

  5. Java, jQuery, Bootstrap, JDBC, Oracle DB
