프로젝트 기간 2020년 8월 28일~2020년 9월 15일
프로젝트 팀 개인프로젝트
활용한 기술 HTML, CSS, JavaScript

  1. 프로젝트 개요

  2. 뱁새에게 바치는 헌정 페이지를 구현하는 개인 프로젝트입니다.
  3. 바닐라 자바스크립트, 순수 CSS 활용으로만 반응형 웹을 구현해보기 위한 미니 프로젝트입니다.
  4. 반응형 웹 디자인이 적용된 정적 프론트 페이지입니다.
  5. 구현된 페이지는 netlify를 통해 배포되었으며, 이곳을 클릭하여 접근이 가능합니다.

Page 1

  1. 반응형 웹 디자인

  2. 사용자 기기 화면 사이즈에 따라 요소 크기와 배치를 달리 하도록 반응형으로 디자인 했습니다.
  3. @media 태그를 사용하여 세가지 화면 사이즈로 구분하여 구현했습니다.
    1. 모바일 기기(max-width: 1000px)
    2. 태블릿 PC(min-width: 1001px)
    3. PC(min-width: 1481px)

Page 2

  1. 리스트 메뉴 / 햄버거 메뉴

  2. 사용자 기기의 화면 사이즈에 따라 메뉴 구성을 리스트 메뉴에서 햄버거 메뉴로 변경되게 설정했습니다.

Page 3

  1. 페이지 언어 전환

  2. 버튼 클릭시 화면의 구성 언어가 변경되게 구현했습니다.
  3. 이번 프로젝트에서는 한국어와 영어 요소를 함께 페이지에 포함시킨 이후 display 설정을 토글하는 방식으로 구현했습니다.
